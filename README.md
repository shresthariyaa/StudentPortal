# Student Portal

## Project Overview
A web-based portal to manage students and courses efficiently. Built with Flask, SQLAlchemy, and Bootstrap. Users can register, login, view courses, enroll, and manage profiles. Admins can manage students and courses.
## Features
- User registration & login
- Profile management
- Dashboard statistics (total students)
- CRUD operations on students & courses
- Course enrollment
- Responsive design for mobile & desktop

## Technology Stack
- **Frontend:** HTML, CSS, Bootstrap, JavaScript
- **Backend:** Python, Flask, Jinja2
- **Database:** SQLite with SQLAlchemy
- **Version Control:** Git & GitHub
## How to Run
1. Clone the repo: `git clone https://github.com/shresthariyaa/StudentPortal.git`
2. Create virtual environment: `python3 -m venv venv`  
   Activate it: `source venv/bin/activate` (Mac/Linux) or `venv\Scripts\activate` (Windows)
3. Install dependencies: `pip install -r requirements.txt`
4. Run the app: `python app.py`
5. Open browser at `http://127.0.0.1:5000`
